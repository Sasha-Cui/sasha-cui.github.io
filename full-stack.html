About the working group

We go beyond theory with **minimal working examples (MWEs)** and short **showcases** that demonstrate “the art of the possible” — customizations, applied demos, and advanced usage patterns. Outputs live in a public GitHub repo and a course-style website.

Supported by **WTI GPUs**. Open to CS, ECE, S&DS PhD students and interested undergraduates.

Cadence & outputs

  • Biweekly meetings (short talks, live demos).
  • Onboarding tutorial(s) for newcomers to LLMs.
  • Shared **GitHub** with MWEs, slides, and notes.
  • **Builder Track** contributors listed as co-authors on the write-up/paper.

Builder Track (core)

Contribute MWEs, tutorials, and a brief “beyond-basics” showcase. Present at least once. PRs to the repo. Expected ~2–4 hrs per two weeks.

  • Hands-on with real packages
  • Short demo talk(s)
  • PRs/docs to GitHub
  • Co-authorship on group write-up
Apply to Builder Track

Explorer Track (non-core)

Attend, learn, and run stacks locally/HPC. Great for interview prep and practical fluency.

  • Biweekly sessions
  • Run MWEs yourself
  • Industry-oriented skills
  • Optional lightning talks
