Becoming Full-Stack AI Researchers

A fortnightly working group covering the entire LLM stack (training, post-training, serving, agents, evaluation, datainfra, interpretability, and safety) with dedicated GPUs provided by Wu Tsai Institute.

Timeline & Outputs

  • Fortnightly meetings (short talks, live demos).
  • Onboarding tutorials for newcomers to LLMs.
  • Shared GitHub Repository with MWEs, slides, and notes.
  • Builder Track contributors listed as co-authors on the tutorial paper.

Builder Track (up to 15 participants)

Contribute MWEs, tutorials, and a brief “beyond-basics” showcase. Present at least once. PRs to the repo.

  • Serve as the local experts on selected topics.
  • Prepare short demo talks covering fundamentals and advanced features
  • Contribute to the GitHub Repo
  • Co-authorship on group write-up
Apply to Be a Builder

Explorer Track (up to 100 participants)

Attend, learn, and run AI frameworks. Great for interview prep and practical fluency.

  • Run MWEs yourself
  • Industry-oriented skills
  • Optional lightning talks
Apply to Be an Explorer

Serving

Eval & Data

lm-eval, Terminal Bench.
PyArrow & HF Datasets.

Infra

Slurm, Docker/Apptainer, Weights & Biases.
